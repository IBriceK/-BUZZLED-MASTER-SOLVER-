📖 How to use this solver (click to expand)
  1. Open Buzzled and select Master mode.
  2. Look at the 21 pins around the hex grid. Each pin has a color (yellow/black) and a number.
  3. The pins are grouped into 3 directions:
    • Top pins (7) — the pins along the top edge, from upper-left corner to upper-right corner
    • Right + Bottom-right pins (7) — going down the right side, then along the bottom-right edge
    • Left + Bottom-left pins (7) — going up the left side, then along the bottom-left edge (entered right-to-left)
  4. For each pin, set its color (Yellow/Black) and number in the inputs below.
  5. Click Solve. The solution grid shows each hex as Yellow (🟡) or Black (⚫), row by row from top to bottom.
